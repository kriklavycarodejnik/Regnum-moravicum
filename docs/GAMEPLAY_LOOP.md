# Regnum Moravicum — herný loop (hráčsky kanon)

## Kto si
Si **Mojmír II.**, knieža **Veľkej Moravy**, rok **902**.
Dynastia Mojmírovcov drží ríšu; susedia (Francia, Bavorsko, Maďari…) ju ohrozujú.

## Ako vyhrať / prehrať
| | Podmienka |
|--|----------|
| **Víťazstvo** | Prežiť s aspoň 1 župou a živou dynastiou do **roku 1000** (alebo neskoršie prestížne/teritoriálne ciele) |
| **Prehra** | Strata všetkých žúp · vymretie dynastie · bankrot |

## Jadro loopu (každý mesiac)
1. **Pozri stav** — zdroje, mapa, ciele vpravo  
2. **Voliteľne konaj** — diplomacia, armády, výber župy  
3. **Stlač „Ďalší mesiac“** — simulácia (ekonomika, AI, eventy)  
4. **Ak príde udalosť** — vyber 1 z 2 volieb  
5. Opakuj do 1000 / konca  

**Primárna akcia hry = „Ďalší mesiac“.**  
Devín 907 a Skirmish sú **scénárové / cvičné bitky**, nie povinný každý ťah.

## Fázy kampane (orientácia)
| Roky | Fáza | Čo robiť |
|------|------|----------|
| 902–906 | **Konsolidácia** | Zbieraj zlato/jedlo, sleduj lojalitu, spoznaj mapu |
| 907 | **Kríza Maďarov** | Bitka pri Devíne — historická skúška (scénár) |
| 908–960 | **Prežitie** | Diplomacia, armády, eventy rady |
| 960–1000 | **Legitimita** | Prestíž, viera, držanie žúp do milénia |

## UI mapa → hráč
- **Mapa** = tvoje župy (klik = detail + ilustrácia)  
- **Hero vpravo** = čo práve sleduješ (župa / bitka / dvor)  
- **Ciele** = čo máš teraz dosiahnuť  
- **Ďalší krok** = jedna odporúčaná akcia
